# Techstack used 
- React Js using Vite
- lucide-react for icons
- Tailwind Css for styling

## How to Run the Project Locally
```bash
git clone https://github.com/hm-aditya/positivus.git
cd positivus
npm install
npm run dev
```
The app will be available at:
http://localhost:5173

Live deployed link :
https://positivus-one-ruddy.vercel.app/
